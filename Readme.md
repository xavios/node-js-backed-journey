## Node JS Crash course

Node.js is an open source JS runtime (programming environment, that runs other programs).
It uses the V8 rendering engine coming from Chrome. Build with C++.
Mostly used for developing server side and networking applications and APIs.

Node.js is not good for CPU intensive applications.

Node.js comes with a REPL.
